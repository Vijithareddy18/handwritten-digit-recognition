# Handwritten Digit Recognition

## Overview
A deep learning-based handwritten digit recognition system developed using Python, TensorFlow, Keras, and OpenCV. The model identifies handwritten digits from images through image preprocessing, segmentation, and classification.

## Features
- Handwritten digit recognition
- Image preprocessing and segmentation
- CNN-based classification
- Interactive prediction correction
- 95%+ accuracy on the MNIST dataset

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Matplotlib

## Results
Achieved over 95% accuracy in recognizing handwritten digits and successfully classified digits from custom images.
